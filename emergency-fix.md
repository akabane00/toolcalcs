Google Search Console에서 toolcalcs.com 속성으로 아래 작업을 긴급 처리해줘.

## 작업 1: 사이트맵 재제출
1. 왼쪽 메뉴 "사이트맵" 클릭
2. 기존 sitemap-index.xml 삭제
3. 새로 "sitemap-index.xml" 입력 → 제출
4. 제출 상태 확인

## 작업 2: 핵심 페이지 5개 색인 재요청
URL 검사 → 각 URL 입력 → "색인 생성 요청" 클릭:

- https://toolcalcs.com/blog/how-much-is-30-an-hour-annually/
- https://toolcalcs.com/blog/how-many-ounces-in-a-gallon/
- https://toolcalcs.com/blog/how-many-tablespoons-in-a-cup/

- https://toolcalcs.com/blog/bmi-chart-adults/
