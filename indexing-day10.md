Google Search Console에서 toolcalcs.com 의 아래 URL들에 대해 색인 생성 요청을 해줘.

방법:
1. https://search.google.com/search-console 접속
2. toolcalcs.com 속성 선택
3. 상단 "URL 검사" 입력창에 URL 입력 → Enter
4. 검사 결과가 나오면 "색인 생성 요청" 클릭
5. 처리 완료 후 다음 URL로 반복
6. 할당량 초과 시 중단하고 남은 URL 목록 알려줘
