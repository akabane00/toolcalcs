Google Search Console에서 toolcalcs.com 속성으로 아래 항목들을 확인해줘.

## 1. 실적 보고 (Performance)
왼쪽 메뉴 "실적" 클릭 후:
- 기간: 최근 28일
- 총 클릭수, 총 노출수, 평균 CTR, 평균 게재순위 알려줘
- 상위 검색어 10개 (노출 기준) 알려줘
- 상위 페이지 10개 (노출 기준) 알려줘

## 2. 색인 범위 확인
왼쪽 메뉴 "페이지" 클릭 후:
- "색인 생성됨" 페이지 수
- "색인 생성되지 않음" 페이지 수
- "크롤링됨 - 현재 색인 미생성" 항목 수
- "robots.txt에 의해 차단됨" 항목 수
- "noindex 태그로 제외됨" 항목 수

## 3. 사이트맵 상태
왼쪽 메뉴 "사이트맵" 클릭 후:
- 사이트맵 제출 상태, 마지막 읽은 날짜, 발견된 URL 수 알려줘

## 4. Core Web Vitals
왼쪽 메뉴 "코어 웹 바이탈" (또는 "페이지 경험") 클릭 후:
- 모바일/데스크톱 상태 (양호/개선필요/불량) 알려줘

각 항목의 결과를 알려줘.
